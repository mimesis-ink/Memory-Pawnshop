# 视觉素材说明

本目录用于存放《记忆当铺》项目的视觉概念设计和参考资料。

## 目录结构

```
assets/
├── concepts/          # 概念设计
│   ├── characters/   # 角色设计
│   ├── scenes/       # 场景设计
│   ├── props/        # 道具设计
│   └── vfx/          # 特效概念
└── references/       # 参考资料
    ├── mood_boards/  # 情绪板
    ├── color_schemes/# 色彩方案
    └── inspiration/  # 灵感来源
```

## 参考风格

### 色彩参考
- **冷色调**：《银翼杀手2049》
- **暖色调**：《她》(Her)
- **霓虹美学**：《攻壳机动队》

### 场景参考
- **未来都市**：《银翼杀手》系列
- **科技空间**：《黑镜》
- **温馨家庭**：现实主义风格

### 特效参考
- **全息投影**：《星球大战》
- **记忆可视化**：《盗梦空间》
- **粒子效果**：《奇异博士》

## 注意事项

1. 所有设计需符合2087年未来世界设定
2. 保持视觉风格统一性
3. 平衡科幻感与真实感
4. 考虑实际拍摄的可行性
5. 控制制作成本

## 提交规范

提交概念设计时，请遵循以下命名规范：

```
[类别]_[名称]_[版本号].[格式]

例如：
character_linsheng_v1.jpg
scene_pawnshop_v2.png
prop_helmet_v1.psd
```
