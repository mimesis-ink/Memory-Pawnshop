## 故事梗概

### 主要故事线（90章）

未来世界，记忆可以像器官一样买卖。穷人卖掉快乐换钱，富人购买快乐填补空虚。

男主林深是一名"记忆提取师"，为了救重病的妹妹，他不得不卖掉自己最珍贵的一段初恋记忆。然而手术后他发现，自己口袋里有一张带血的巨额支票，而所有线索都指向——他刚刚卖掉的记忆，并不是"初恋"，而是他"目击了一场针对顶级富豪的谋杀"。

现在，富豪要来灭口，而他必须找回那段被删除的记忆才能活命。

在2087年的未来世界，记忆交易已成为合法产业。记忆提取师林深为救妹妹林雨，被迫卖掉珍贵记忆，却在醒来后发现口袋里有张带血的500万支票。

随着调查深入，林深发现自己卷入了一场涉及记忆产业巨头陈煜的谋杀案。在女调查员苏晚的帮助下，他们逐步揭开了一个庞大的阴谋：

**第一层真相**（1-30章）：陈煜通过购买目击者记忆来掩盖罪行，背后是记忆黑市和警方腐败的勾结。

**第二层真相**（31-60章）：存在一个名为"永生会"的跨国组织，推行"创世纪"AI计划——通过标准化记忆来统一控制人类意识。多个关键人物拥有克隆体，包括陈煜和赵医生。赵医生将自己的意识上传到数字世界，成为关键变量。

**第三层真相**（61-90章）：林深发现自己的父亲周明就是记忆技术的创始人，整个阴谋的源头。父亲创建技术的初衷是为了人类福祉，但技术被滥用后引发了全球危机。三个赵医生、多个陈煜克隆体、全球记忆觉醒运动、海底与极地的秘密基地——所有线索最终汇聚到父子对决。

在意识层面的最终战斗中，林深必须在摧毁父亲的遗产与保护人类自由意志之间做出选择。而苏晚的真实身份，也与林深被删除的那段"初恋记忆"紧密相关。

**最终**：林深与苏晚携手，打破了记忆控制的桎梏，开启了一个新的纪元——记忆技术依然存在，但人类重新掌握了选择的权利。

## 主要人物

### 核心人物

#### 林深（男主，28岁）
- **职业**：记忆提取师
- **性格**：理性、冷静、擅长心理分析、坚韧不屈
- **背景**：为了照顾重病的妹妹林雨，工作压力大，经济困难
- **核心矛盾**：职业道德与亲情之间的挣扎
- **角色发展**（90章）：从普通提取师 → 被动卷入阴谋 → 主动反抗的斗士 → 面对父亲真相的儿子 → 新世界的见证者

#### 苏晚（女主，26岁）
- **职业**：记忆当铺调查员
- **性格**：聪明、独立、有正义感、坚强勇敢
- **背景**：表面是记忆当铺的员工，实际是调查记忆黑市的卧底；林深的女朋友
- **秘密**：她购买过别人的记忆，对林深有莫名的熟悉感；五年前遭遇车祸，部分记忆被重构
- **角色发展**（90章）：从卧底调查员 → 林深的盟友 → 揭露自身身份谜团 → 抉择真相与爱情 → 共同战斗的伴侣

#### 林雨（妹妹，18岁）
- **职业**：学生
- **疾病**：神经退行性疾病，需要昂贵的记忆修复手术
- **性格**：善良、坚强、不愿成为哥哥的负担
- **角色发展**（90章）：从被保护者 → 多次成为人质 → 觉醒的见证者

### 主要反派

#### 陈煜（初期反派，45岁）
- **职业**：顶级富豪，记忆产业巨头
- **性格**：冷酷、狡猾、掌控欲强
- **秘密**：通过购买和删除记忆来掩盖自己的罪行；拥有多个克隆体
- **角色发展**（90章）：第10章自杀，但其克隆体在后续章节中继续作为对手出现

#### 赵医生（复杂角色，50岁）
- **职业**：记忆当铺主任医师
- **性格**：表面和善，实际利益至上，但后期经历觉醒
- **角色**：知道许多秘密，是关键人物
- **角色发展**（90章）：存在三个版本——原版赵医生、被控制的赵医生、上传数字意识的赵医生；最终在第69章做出牺牲性选择，第60章忏悔

#### 周明（最终反派，林深的父亲，55岁）
- **职业**：记忆技术的创始人，隐藏身份多年
- **性格**：天才科学家，理想主义者，但手段冷酷
- **秘密**：创建了记忆技术体系，是整个阴谋的源头
- **角色发展**（90章）：第25章身份暗示，第71章后全面展开，第89章与林深最终对决

### 其他重要角色

#### 李明（永生会领袖）
- 出现于第30-50章，推动"创世纪"计划的关键人物

#### 韩世杰（警局长）
- 第20-45章的重要角色，双重身份，与永生会有关联

#### 宋文博（记忆产业举报人）
- 被陈煜谋杀的受害者，其死亡是整个故事的导火索
- 在后期章节中以克隆体形式回归（第71章）

#### 沈微（技术专家）
- 宋文博的学生，林深的研究伙伴
- 技术专家，与李明、赵启明同期

#### 申薇（林深的初恋？）
- 林深记忆中被删除的神秘女孩
- 真实身份与苏晚有复杂关联
