# 记忆当铺：我卖掉了杀你的记忆

**类型**：网络小说  
**版本**：90章网络小说（详见 `script/chapters/` 目录）  
**文体**：科幻、悬疑、惊悚  
**核心概念**："贫穷限制了想象力，也限制了你拥有美好记忆的权利。"

---

## 故事梗概

### 主要故事线（90章）

未来世界，记忆可以像器官一样买卖。穷人卖掉快乐换钱，富人购买快乐填补空虚。

男主林深是一名"记忆提取师"，为了救重病的妹妹，他不得不卖掉自己最珍贵的一段初恋记忆。然而手术后他发现，自己口袋里有一张带血的巨额支票，而所有线索都指向——他刚刚卖掉的记忆，并不是"初恋"，而是他"目击了一场针对顶级富豪的谋杀"。

现在，富豪要来灭口，而他必须找回那段被删除的记忆才能活命。

在2087年的未来世界，记忆交易已成为合法产业。记忆提取师林深为救妹妹林雨，被迫卖掉珍贵记忆，却在醒来后发现口袋里有张带血的500万支票。

随着调查深入，林深发现自己卷入了一场涉及记忆产业巨头陈煜的谋杀案。在女调查员苏晚的帮助下，他们逐步揭开了一个庞大的阴谋：

**第一层真相**（1-30章）：陈煜通过购买目击者记忆来掩盖罪行，背后是记忆黑市和警方腐败的勾结。

**第二层真相**（31-60章）：存在一个名为"永生会"的跨国组织，推行"创世纪"AI计划——通过标准化记忆来统一控制人类意识。多个关键人物拥有克隆体，包括陈煜和赵医生。赵医生将自己的意识上传到数字世界，成为关键变量。

**第三层真相**（61-90章）：林深发现自己的父亲周明就是记忆技术的创始人，整个阴谋的源头。父亲创建技术的初衷是为了人类福祉，但技术被滥用后引发了全球危机。三个赵医生、多个陈煜克隆体、全球记忆觉醒运动、海底与极地的秘密基地——所有线索最终汇聚到父子对决。

在意识层面的最终战斗中，林深必须在摧毁父亲的遗产与保护人类自由意志之间做出选择。而苏晚的真实身份，也与林深被删除的那段"初恋记忆"紧密相关。

**最终**：林深与苏晚携手，打破了记忆控制的桎梏，开启了一个新的纪元——记忆技术依然存在，但人类重新掌握了选择的权利。

#### 李婉（李明的妹妹，已故，享年12岁）
- **关系**：李明的妹妹，比李明小8岁（2065年-2077年）

  - 症状进展：轻微记忆混乱 → 记忆流失和认知衰退 → 神经功能全面衰退
  - 治疗尝试：当时记忆技术尚不成熟（2077年），最终因技术限制和伦理禁令未能实施救治
- **背景**：

  - 兄妹二人相依为命
  - 11岁时发病，李明当时正在大学攻读神经科学
  - 李明为救妹妹，提前接触记忆修复技术，开始跟随宋文博学习
  - 病情恶化期间，李明曾带她参观实验室，给她看记忆可视化技术
  - 对哥哥说过："哥哥，总有一天，这个技术会救很多人。"

  - "如果记忆技术还不成熟，就不要勉强"
  - "我宁愿死，也不想看到你为了救我而伤害别人"
  - "答应我，好好活下去，用正确的方式让技术造福更多人"
  - 李明含泪答应了，但最终违背了这个承诺
- **对李明的影响**：
  - 初期：妹妹的死成为李明投身记忆技术研究的初始动力
  - 理想化：最初希望通过技术救更多像妹妹一样的病患
  - 扭曲期：长期的挫折和失望让他逐渐偏离妹妹的遗愿
  - 极端化：最终发展出偏执的"永生会"理念，认为只有永生才能避免失去
  - 自我欺骗：声称"都是为了她"，但实际已背离妹妹的温柔与善良
- **在故事中的作用**：
  - 情感锚点：李明人性中最后的柔软之处
  - 道德标尺：提醒观众李明曾经的纯真和善良
  - 转折关键：林雨提起李婉的遗言，曾让李明短暂动摇（第40章）
  - 悲剧对比：李婉希望技术造福人类，李明却用技术奴役人类
  - 救赎可能：妹妹的记忆可能成为唤醒李明良知的最后希望
- **与林雨的对比**：
  - 两人都是患病的妹妹
  - 李婉死于技术不成熟，林雨得救于技术进步
  - 李婉让李明走向疯狂，林雨让林深保持人性
  - 林雨的存在提醒李明妹妹曾经的模样（第40章）
- **象征意义**：
  - 纯真的丧失：李婉的死象征李明失去了心中最纯净的部分
  - 技术的双刃剑：她因技术不成熟而死，也成为技术发展的催化剂
  - 爱与疯狂的界限：深爱妹妹的李明最终因爱而疯狂
  - 遗愿的背叛：故事主题之一——初心与现实的残酷对比
