## 🎯 使用建议

### 决策者/出版方
1. **第一步**：阅读 [执行摘要](EXECUTIVE_SUMMARY.md) - 5分钟了解核心结论
2. **第二步**：查看 [书名推荐](book_title_recommendations.md) - 确定书名和市场定位
3. **第三步**：参考 [平台分析报告](platform_publishing_analysis.md) - 制定发布策略

### 新手作者
1. **第一步**：完整阅读 [写作风格指南](writing_style_guide.md)，理解项目的核心风格
2. **第二步**：学习 [章节创作流程](chapter_writing_workflow.md)，掌握完整创作流程
3. **第三步**：将 [写作快速参考](quick_reference.md) 保存为书签，随时查阅

### 经验作者
1. **日常创作**：使用 [写作快速参考](quick_reference.md) 进行快速查阅
2. **遇到问题**：查阅 [章节创作流程](chapter_writing_workflow.md) 的"常见问题解决"部分
3. **风格校准**：定期回顾 [写作风格指南](writing_style_guide.md)，确保风格一致性

### 编辑审校
1. 使用 [质量检查清单](chapter_writing_workflow.md#质量检查清单) 审核章节
2. 参照 [写作禁忌与要求](writing_style_guide.md#写作禁忌与要求) 进行质量把关
3. 确保所有章节符合风格规范

## 📋 快速风格检查

在提交章节前，确保符合以下标准：

✅ **风格定位**
- [ ] 赛博朋克 + 黑色电影风格
- [ ] 压抑、冷峻、克制的基调
- [ ] 高科技与低生活的对比

✅ **结构规范**
- [ ] 1800-2000字
- [ ] 有"黄金三秒"开场
- [ ] 有"结尾钩子"
- [ ] 2-3个场景

✅ **核心技法**
- [ ] 环境描写隐喻人物内心
- [ ] 对话简练有潜台词
- [ ] 多重感官细节
- [ ] 冷色调（蓝、灰、黑）

✅ **避免禁忌**
- [ ] 无过度抒情煽情
- [ ] 无冗长心理描写
- [ ] 无平淡开头结尾
- [ ] 无抽象词汇（"未来感"等）

## 💡 写作理念

> 在这个可以买卖一切的世界，你的文字要像霓虹灯下的雨夜一样，冰冷、压抑，但又隐含着微弱的光——那是人性最后的温度。

### 核心要素
1. **赛博朋克** - 高科技低生活，视觉对比强烈
2. **黑色电影** - 明暗对比，道德灰色地带
3. **环境隐喻** - 以景写情，细节暗示
4. **语言克制** - 简练有力，少即是多
