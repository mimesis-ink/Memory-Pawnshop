## ⚡ 5分钟快速理解

### 故事一句话
穷小子卖记忆救妹妹，却发现卖掉的是一段谋杀目击记忆，现在他必须找回记忆才能活命。

### 核心卖点
1. **开篇即高能**：3秒内抛出悬念（血+500万支票）
2. **贫富对比强烈**：穷人卖快乐，富人买快乐
3. **悬疑层层递进**：每集一个新悬念
4. **爽文反转**：第6集反转（买家是死者），第15集终极反转（人格分裂）

### 情感内核
- **亲情**：为救妹妹不惜一切
- **正义**：揭露记忆黑市
- **成长**：从受害者到行动者

## 🎯 制作重点（必看）

### 黄金场景（必须精心拍摄）
1. ⭐ **第1集开篇3秒** - 成败关键，务必有冲击力
2. ⭐ **第5集拍卖会** - 视觉奇观，展示贫富对比
3. ⭐ **第6集VIP包厢** - 重大反转，慢动作+情绪
4. ⭐ **第15集结局演讲** - 主题升华，情感共鸣

### 必备道具（低成本高效果）
- 后脑记忆接口（LED贴片，<100元）
- 发光记忆存储管（试管+LED，<50元/个）
- 面具（拍卖会用，可租赁）
- 血迹支票（道具，<20元）

### 色彩系统
- 灰色 = 贫穷（林墨出租屋）
- 蓝紫色 = 科技（记忆存储器）
- 金色 = 奢靡（拍卖会）
- 红色 = 危险（犯罪记忆）

## 📅 拍摄周期

- **筹备期**：1周（场景、道具、演员）
- **拍摄期**：2-3周（按场景集中拍摄）
- **后期期**：2-3周（剪辑、特效、音效）
- **总计**：5-7周

---

## 🎬 拍摄顺序建议

**按场景拍，不按集数拍**：

1. **Week 1**: 出租屋场景（第1、2、4、9集）
2. **Week 2**: 当铺场景（第2、3、7-8集）
3. **Week 3**: 酒店场景（第4、5、6集）
4. **Week 4**: 其他场景（医院、贫民窟、街头）

## ⚠️ 常见问题

### Q1: 特效会不会很贵？
**A**: 不会。主要特效是全息屏幕和记忆画面处理，可用后期软件完成，预算5000-15000元。

### Q2: 需要专业演员吗？
**A**: 不一定。可用短视频达人或新人演员，关键是气质合适、演技到位。

### Q3: 拍摄难度大吗？
**A**: 中等。场景简单（室内为主），动作戏少，主要考验剧情节奏和演员表演。

### Q4: 会不会过不了审核？
**A**: 已考虑审核风险。暴力画面用光影暗示，明确立场（记忆黑市=犯罪），强调正能量（亲情、正义）。
