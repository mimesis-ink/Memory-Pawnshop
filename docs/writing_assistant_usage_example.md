# AI写作助手使用示例

本文档演示如何使用 **[AI写作助手系统提示词](writing_assistant_prompt.md)** 来辅助创作《记忆当铺》的章节内容。

## 使用方法

### 步骤一：加载系统提示词

将 `writing_assistant_prompt.md` 的内容作为系统提示（System Prompt）输入给AI助手。这将使AI助手：
- 理解赛博朋克和黑色电影的风格要求
- 掌握通过环境描写隐喻人物内心的技巧
- 熟悉《记忆当铺》的世界观和人物设定
- 遵循"黄金三秒"和"结尾钩子"的创作规范

### 步骤二：提供创作需求

明确告诉AI助手你需要创作什么内容，例如：

```
请根据《记忆当铺》的设定，创作第X章的开场"黄金三秒"，要求：
- 场景：林深在记忆黑市调查线索
- 气氛：紧张、危险
- 风格：赛博朋克 + 黑色电影
- 字数：50-100字
```

### 步骤三：审核与优化

AI生成内容后，对照以下检查清单进行审核：

**风格检查**：
- [ ] 是否符合赛博朋克美学（高科技低生活、霓虹灯、未来城市）
- [ ] 是否体现黑色电影风格（压抑、冷峻、道德灰色）
- [ ] 是否通过环境描写隐喻人物内心
- [ ] 语言是否简练克制，避免过度抒情

**结构检查**：
- [ ] "黄金三秒"是否在3秒内抓住读者
- [ ] 主体内容是否快速推进，无拖沓
- [ ] "结尾钩子"是否制造悬念
- [ ] 总字数是否在1800-2000字范围

**设定检查**：
- [ ] 是否符合2087年的世界观
- [ ] 人物行为是否符合角色性格
- [ ] 记忆技术描写是否准确
- [ ] 社会阶级描写是否到位

## 示例对比

### 示例1：开场"黄金三秒"

**需求**：创作第5章的开场，林深进入记忆黑市

**不符合风格的版本**（平淡、缺乏画面感）：
```
林深来到了记忆黑市。这里很脏很乱，有很多人在交易记忆。他感到很紧张，因为这里很危险。
```

**符合风格的版本**（赛博朋克+黑色电影）：
```
记忆黑市藏在城市的最底层。林深走下锈迹斑斑的楼梯，每一步都伴随着金属的呻吟。墙上涂满了荧光涂鸦，在黑暗中发出诡异的蓝光。空气里弥漫着消毒水和腐败的气味——那是非法记忆手术的痕迹。他握紧口袋里的那张血色支票，手心冒出冷汗。在这个无法无天的地方，一旦被人发现他身上有500万，他可能永远走不出去。
```

**分析**：
- ✅ 视觉元素：锈迹、荧光涂鸦、蓝光
- ✅ 氛围营造：黑暗、危险、无法无天
- ✅ 环境隐喻：脏乱差的环境反映社会底层的绝望
- ✅ 人物心理：通过"冷汗"暗示紧张，不直接说"他很紧张"

### 示例2：结尾"钩子"

**需求**：为第8章设计结尾钩子，林深发现了关键线索

**不符合风格的版本**（无悬念）：
```
林深找到了线索，决定明天去调查。他回家睡觉了。
```

**符合风格的版本**（悬念+危机）：
```
就在林深准备离开时，一个女人的声音从黑暗中传来："你在找什么？"他猛地转身，看到一个穿着黑色风衣的女人站在门口，逆光中看不清她的脸。林深下意识地往后退，却发现身后的墙上投射着一个影子——有人一直在跟踪他。女人轻笑了一声："林深，你卖掉的那段记忆，现在有很多人想要。包括我。"
```

**分析**：
- ✅ 悬念制造：神秘女人的身份
- ✅ 危机升级：被跟踪，处境危险
- ✅ 视觉描写：逆光、影子
- ✅ 对话张力："包括我"暗示冲突

### 示例3：环境描写隐喻

**需求**：描写林深内心的绝望

**不符合风格的版本**（直接描述情绪）：
```
林深感到很绝望。他觉得自己陷入了困境，不知道该怎么办。他的内心充满了恐惧和不安。
```

**符合风格的版本**（环境隐喻）：
```
窗外的雨下得越来越大，雨水顺着玻璃流下，将城市的霓虹灯拉成无数条血痕。林深站在窗前，看着街上匆匆而过的行人——每个人都有自己的目的地，只有他，迷失在这座城市的迷宫里。记忆当铺的霓虹招牌在雨夜中闪烁，红蓝交替的光像是在嘲笑他——你卖掉了记忆，却买不回安全。
```

**分析**：
- ✅ 天气隐喻：大雨象征困境
- ✅ 视觉意象：霓虹灯被雨水拉成血痕
- ✅ 对比手法：他人有目的地，他迷失
- ✅ 反讽：卖记忆换不来安全

## 常见问题

### Q1：如何避免写得太压抑？
A：虽然基调是压抑冷峻，但要保持适当的节奏变化。在紧张场景后，可以加入短暂的平静时刻（如林深与妹妹的对话），但仍要保持克制，不过度煽情。

### Q2：如何把握"黄金三秒"的长度？
A：50-100字是指中文字符数。通常是2-3个短句，或1-2个长句。开场要立即切入核心冲突或悬念，不做铺垫。

### Q3：环境描写会不会显得啰嗦？
A：关键是让环境描写"有功能性"——必须隐喻人物心理或推进情节。如果一段环境描写删掉后不影响理解，那就是冗余的。

### Q4：对话如何体现"黑色电影"风格？
A：
- 简洁有力，少用"他说""我想"等标签
- 多用潜台词，不把话说满
- 适当使用冷幽默和讽刺
- 避免长篇大论的解释

### Q5：如何检验是否符合赛博朋克风格？
A：检查是否包含这些元素：
- 高科技设备（记忆晶片、全息投影、神经接口）
- 社会批判（贫富差距、人性异化）
- 视觉元素（霓虹灯、雨夜、破败建筑）
- 科技伦理困境

## 进阶技巧

### 技巧1：色彩运用
- **冷色调**（蓝、灰、黑）：现实世界，营造压抑感
- **暖色调**（金、橙）：记忆世界，对比温暖与冰冷
- **特殊色彩**：血红（危险）、荧光绿（科技）、霓虹（虚假繁华）

### 技巧2：光影对比
- 强烈明暗对比（chiaroscuro）：象征道德灰色地带
- 阴影中的人物：神秘、危险
- 逆光：看不清真相

### 技巧3：物象象征
- 雨水：洗不掉的罪恶、持续的压抑
- 镜子：自我认同、真假难辨
- 玻璃：脆弱、透明却隔绝
- 烟雾：真相的模糊

### 技巧4：节奏控制
- 紧张场景：短句、快节奏
- 情感场景：适当放慢，但仍保持克制
- 反转时刻：突然的句式变化

## 总结

使用AI写作助手时，记住：
1. **风格优先**：赛博朋克+黑色电影是底色
2. **隐喻为王**：不直接说情绪，用环境说话
3. **克制表达**：压抑冷峻，不煽情不啰嗦
4. **结构严格**："黄金三秒"+"结尾钩子"必不可少
5. **设定忠实**：符合世界观，人物不OOC

只有同时满足这五点，才能创作出符合《记忆当铺》标准的高质量内容。
