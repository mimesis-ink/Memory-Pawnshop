## 创作前准备

### 步骤1：确定章节主题

在开始写作前，明确以下内容：

#### 剧情定位
- [ ] 本章在整体故事中的位置（第几章？第几阶段？）
- [ ] 本章需要完成的剧情任务（推进主线？揭露秘密？）
- [ ] 本章的前置剧情回顾（上一章发生了什么？）
- [ ] 本章的后续影响（为下一章埋下什么伏笔？）

#### 核心冲突
- [ ] 本章的主要冲突是什么？（内心冲突？人际冲突？生存危机？）
- [ ] 冲突如何升级？
- [ ] 冲突如何（部分）解决或转化？

#### 情感点
- [ ] 本章的情感基调（紧张？悲伤？绝望？微光？）
- [ ] 人物情感变化弧线
- [ ] 读者期待获得的情感体验

### 步骤2：检查人物与世界观

#### 人物一致性
- [ ] 角色行为符合性格设定
- [ ] 角色对话符合语言风格
- [ ] 角色关系发展合理
- [ ] 注意人物成长弧线

#### 世界观一致性
- [ ] 技术设定符合2087年背景
- [ ] 社会阶层描写准确
- [ ] 记忆交易细节合理
- [ ] 环境描写符合赛博朋克美学

### 步骤3：规划场景结构

每章建议**2-3个场景**，确保节奏紧凑：

```
【推荐结构】
场景1（600-700字）：开场冲突，抛出问题
场景2（600-700字）：冲突深化，推进剧情
场景3（600-700字）：转折或悬念，引向下章

【总计】1800-2100字
```

## "黄金三秒"开场技巧

### 原则

**在开头50-100字内立即抓住读者**

读者在开始阅读的前3秒就会决定是否继续。因此，开场必须：
- ✅ 立即建立冲突或悬念
- ✅ 调动感官（视觉、听觉、触觉）
- ✅ 营造赛博朋克氛围
- ❌ 避免冗长的背景说明
- ❌ 避免平淡的日常描写

### 五种开场方式

#### 1. 冲突开场

直接展示危机或对立状态。

**示例**：
```
剧烈的偏头痛像电钻一样凿穿了太阳穴。

林深猛地坐起，一张带血的支票从他汗湿的衬衫口袋里滑落，
在无菌的白色地板上格外刺眼。
```

**适用场景**：危机章节、转折点、高潮部分

#### 2. 悬念开场

抛出引人好奇的问题或异常现象。

**示例**：
```
赵医生的办公室空无一人。

桌上的咖啡还在冒着热气，全息屏幕没有关闭，
数据流在半空缓缓流淌——仿佛主人只是离开了一分钟。

但林深知道，赵医生已经失踪了三天。
```

**适用场景**：调查章节、谜题展开、秘密揭露

#### 3. 动作开场

激烈的场景，快节奏的行动。

**示例**：
```
苏晚侧身避开激光刀，反手用记忆读取器的探针
刺入了杀手的麻筋。

对方惨叫一声，摔倒在积水中。
霓虹灯的倒影在水面破碎，像炸裂的血花。
```

**适用场景**：追逐戏、战斗场面、紧急逃脱

#### 4. 对话开场

震撼性台词，信息量大。

**示例**：
```
"你卖掉的不是初恋记忆，林深。"

电话那头，赵医生的声音低沉而沙哑，
"是一场谋杀。"

林深的手指僵住了。
```

**适用场景**：真相揭露、关键信息、人物对决

#### 5. 反常开场

违背常理的场景，制造不安感。

**示例**：
```
记忆当铺今天不营业。

这很奇怪。在这个城市，记忆交易从不停歇，
就像人们的贪婪和绝望从不停歇。

但今天，招牌的蓝光熄灭了。
```

**适用场景**：平静前的暴风雨、异常事件、气氛营造

### 开场检查清单

写完开场后，检查以下要点：

- [ ] 前50-100字是否立即抓人？
- [ ] 是否调动了感官（至少2种）？
- [ ] 是否营造了赛博朋克氛围？
- [ ] 是否符合人物性格和剧情逻辑？
- [ ] 读者是否会想继续读下去？

## 章节主体构建

### 场景推进原则

#### 每个场景都要推动剧情
- ❌ 避免：纯粹的日常描写、冗长的回忆、无意义的对话
- ✅ 应该：揭示新信息、深化冲突、推动行动、展现人物成长

#### 场景转换要流畅
```
【场景1结束】
林深挂断电话，手指在颤抖。

【场景2开始（时空转换）】
半小时后，他站在记忆当铺门口。
```

#### 环境描写隐喻心境

**不好的写法**：
```
林深很紧张，心跳加速。
```

**好的写法**：
```
提取室的灯光忽明忽暗，嗡鸣声像是远处的雷声，
一点点逼近。林深的衬衫领口湿了一圈。
```

### 对话写作技巧

#### 简练有力
- ❌ 冗长：林深说："我觉得这件事情可能没有那么简单，我们需要仔细调查一下。"
- ✅ 简练：林深说："有问题。"

#### 有潜台词
```
"你还记得我吗？"苏晚问。

林深沉默了两秒，"应该记得。"

"应该？"

"记忆这东西，谁说得准。"
```

#### 对话推动剧情
```
"陈煜死了。"

"什么时候？"

"今晚。他开枪自杀了。"

林深的手指蜷紧，"还是被自杀？"

苏晚没说话。这就是答案。
```

### 节奏控制

#### 紧张场景：短句 + 快节奏
```
脚步声。
越来越近。
林深屏住呼吸。

门把手转动了。
```

#### 缓和场景：长短句结合
```
林雨靠在病床上，脸色苍白，但笑容依然温柔。

"哥，我没事。真的。"

林深没说话。他只是看着妹妹，
心里某个地方在慢慢碎裂。
```

### 感官细节

#### 视觉（最常用）
- 颜色：蓝光、灰色、霓虹
- 光影：阴影、闪烁、反射
- 质感：锈迹、裂纹、积水

#### 听觉（营造氛围）
- 近：电流声、键盘声、呼吸声
- 中：脚步声、对话声、机械声
- 远：警笛、雨声、引擎声

#### 嗅觉（唤起记忆）
- 化学味：消毒水、臭氧
- 衰败味：霉味、锈味、血腥味
- 机械味：润滑油、橡胶、金属

#### 触觉（强化体验）
- 温度：冰冷的金属、湿冷的空气
- 质地：粗糙的墙面、光滑的玻璃
- 感受：刺痛、麻木、颤抖

## "结尾钩子"设计

### 原则

**每章结尾必须留下悬念，引发下一章期待**

读者读完本章后，应该：
- ✅ 想知道"然后呢？"
- ✅ 对某个问题感到好奇
- ✅ 为角色命运担忧

### 六种结尾钩子

#### 1. 真相揭露型

揭示一个震撼性的事实。

**示例**：
```
林深翻开文件最后一页，瞳孔骤然收缩。

DNA对比结果：99.7%匹配。

周明，就是他的父亲。
```

#### 2. 危机升级型

局势急转直下，陷入更大危险。

**示例**：
```
苏晚挂断电话，脸色苍白。

"我们被包围了。"

窗外，三十架武装飞行器的灯光，
将整栋大楼照得如同白昼。
```

#### 3. 意外转折型

出现完全意料之外的变化。

**示例**：
```
门打开了。

站在门外的不是陈煜，不是杀手，
而是林深以为已经死了十五年的父亲。

"好久不见，儿子。"
```

#### 4. 悬念问题型

抛出一个需要下章解答的问题。

**示例**：
```
林深盯着那张照片。

照片上，年轻的赵医生和宋文博站在一起，
还有一个人——

但那个人的脸，被人用刀划掉了。

是谁？
```

#### 5. 倒计时型

建立紧迫感和时间压力。

**示例**：
```
全息屏幕上，倒计时开始跳动：
23:59:58
23:59:57
23:59:56

二十四小时后，"创世纪"病毒将全球扩散。
```

#### 6. 情感冲击型

人物或关系的重大变化。

**示例**：
```
林雨闭上眼睛，呼吸越来越微弱。

"哥..."

心电监护仪的线条，慢慢趋于平缓。

然后，一声长鸣。
```

### 结尾检查清单

- [ ] 是否制造了悬念或期待？
- [ ] 是否与开场形成呼应？
- [ ] 是否为下一章留下铺垫？
- [ ] 是否符合本章情感基调？
- [ ] 读者是否会想继续阅读？

## 质量检查清单

### 完成初稿后，逐项检查：

#### 结构检查
- [ ] 字数控制在1800-2000字
- [ ] 有"黄金三秒"开场
- [ ] 有"结尾钩子"
- [ ] 场景数量2-3个
- [ ] 场景转换流畅

#### 风格检查
- [ ] 符合赛博朋克美学（高科技低生活）
- [ ] 符合黑色电影风格（压抑冷峻）
- [ ] 环境描写隐喻人物内心
- [ ] 语言简练克制
- [ ] 冷色调为主

#### 内容检查
- [ ] 人物行为符合性格设定
- [ ] 对话简练有潜台词
- [ ] 世界观设定一致
- [ ] 技术细节合理
- [ ] 时间线准确

#### 技巧检查
- [ ] 运用多重感官（至少2种）
- [ ] 明暗对比（光影描写）
- [ ] 具体细节代替抽象词汇
- [ ] 短句制造紧张感
- [ ] 伏笔合理埋设

#### 禁忌检查
- [ ] 无过度抒情和煽情
- [ ] 无冗长心理描写
- [ ] 无说教式对话
- [ ] 无脱离世界观
- [ ] 无违背人物性格
- [ ] 无平淡开头结尾
- [ ] 无拖沓节奏

## 常见问题解决

### 问题1：开场写不出"黄金三秒"

**解决方案**：
1. 从本章最激烈的冲突入手
2. 从最强的感官刺激入手
3. 从最大的悬念入手
4. 参考五种开场方式，选择最合适的

### 问题2：字数超标（超过2000字）

**解决方案**：
1. 删除不推动剧情的描写
2. 压缩冗长的对话
3. 合并相似的场景
4. 将次要剧情移至下章

### 问题3：字数不足（少于1800字）

**解决方案**：
1. 增加感官细节（但不要为凑字而写）
2. 深化人物内心冲突
3. 添加环境描写隐喻
4. 如果剧情确实简单，保持精简（1500-1700字也可接受）

### 问题4：节奏拖沓

**解决方案**：
1. 删除不必要的铺垫
2. 加快场景转换
3. 用短句加速节奏
4. 每个段落都问自己："这推动剧情了吗？"

### 问题5：环境描写不够赛博朋克

**解决方案**：
1. 增加"高科技 vs 低生活"对比
2. 添加霓虹灯、全息投影等标志元素
3. 强调金属、混凝土的冷硬质感
4. 用蓝色、灰色、黑色的冷色调

### 问题6：人物对话不够有力

**解决方案**：
1. 删除无意义的客套话
2. 每句对话都要有潜台词或推动剧情
3. 用短句代替长句
4. 增加沉默和停顿的运用

### 问题7：结尾缺乏悬念

**解决方案**：
1. 参考六种结尾钩子类型
2. 在结尾前埋设伏笔
3. 留下一个未解答的问题
4. 制造危机或转折

## 创作流程总结

```
【准备阶段】
1. 确定章节主题（剧情、冲突、情感）
2. 检查人物与世界观一致性
3. 规划2-3个场景

【写作阶段】
4. 写"黄金三秒"开场（50-100字）
5. 展开主体内容（1600-1800字）
   - 场景1：建立冲突
   - 场景2：深化冲突
   - 场景3：转折悬念
6. 设计"结尾钩子"（100-200字）

【检查阶段】
7. 质量检查清单逐项核对
8. 修改优化，删繁就简
9. 最终确认字数（1800-2000字）

【完成】
```
