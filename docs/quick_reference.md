## 📋 风格速查表

### 核心风格
- **赛博朋克** + **黑色电影**
- **压抑、冷峻、克制**
- **环境隐喻人物内心**

### 色调关键词
- 🔵 **蓝**：记忆晶片、冷光、雨夜
- ⚫ **灰**：混凝土、金属、阴霾
- ⚫ **黑**：阴影、夜晚、绝望

## ✅ Do's（必须做）

### 开场
✅ 黄金三秒：立即冲突/悬念（50-100字）  
✅ 调动感官：视觉、听觉、嗅觉  
✅ 赛博朋克氛围：霓虹、蓝光、破败  

### 主体
✅ 环境描写隐喻内心  
✅ 对比：高科技 vs 低生活  
✅ 短句制造紧张感  
✅ 对话简练有潜台词  
✅ 每个场景推动剧情  

### 结尾
✅ 结尾钩子：悬念/转折/危机  
✅ 为下章留伏笔  

---

## ❌ Don'ts（禁止做）

❌ 过度抒情煽情  
❌ 冗长心理描写  
❌ 说教式对话  
❌ 平淡开头结尾  
❌ 拖沓节奏  
❌ 抽象词汇（"未来感"、"科技感"）  
❌ 违背人物性格  
❌ 脱离世界观

## 🎨 环境描写公式

### 赛博朋克对比
```
【高科技】 ⟷ 【低生活】
全息广告 ⟷ 破败墙面
精密设备 ⟷ 锈蚀管道
记忆蓝光 ⟷ 积水阴影
```

### 感官细节
- **视觉**：蓝光、霓虹、阴影、反射、锈迹、裂纹
- **听觉**：电流滋滋声、压缩机嗡鸣、警笛、雨声
- **嗅觉**：消毒水、金属锈味、臭氧、霉味

### 示例对照

❌ **不好**：
```
林深很焦虑，心情很糟糕。房间很破旧。
```

✅ **更好**：
```
提取室的灯光忽明忽暗，嗡鸣声像远处的雷声，
一点点逼近。墙角瓷砖发黑，渗出潮湿的霉味。
```

## 💬 对话写作技巧

### 简练原则
❌ 冗长："我觉得这件事情可能没有那么简单，我们需要仔细调查一下。"  
✅ 简练："有问题。"

### 潜台词示例
```
"你还记得我吗？"苏晚问。
林深沉默了两秒，"应该记得。"
"应该？"
"记忆这东西，谁说得准。"
```

### 对话推动剧情
```
"陈煜死了。"
"什么时候？"
"今晚。他开枪自杀了。"
林深的手指蜷紧，"还是被自杀？"
苏晚没说话。这就是答案。
```

## 🎬 开场技巧速查

### 1. 冲突开场
```
剧烈的偏头痛像电钻一样凿穿了太阳穴。
林深猛地坐起，一张带血的支票从口袋滑落。
```

### 2. 悬念开场
```
赵医生的办公室空无一人。
咖啡还在冒热气，但他已经失踪了三天。
```

### 3. 动作开场
```
苏晚侧身避开激光刀，反手用探针刺入杀手麻筋。
对方惨叫倒地，霓虹灯倒影在积水中破碎。
```

### 4. 对话开场
```
"你卖掉的不是初恋记忆，林深。"
电话那头，赵医生声音低沉，"是一场谋杀。"
```

### 5. 反常开场
```
记忆当铺今天不营业。
这很奇怪。记忆交易从不停歇。
但今天，招牌的蓝光熄灭了。
```

## 🎣 结尾钩子速查

### 1. 真相揭露
```
林深翻开文件最后一页，瞳孔骤然收缩。
DNA对比结果：99.7%匹配。
周明，就是他的父亲。
```

### 2. 危机升级
```
苏晚脸色苍白，"我们被包围了。"
窗外，三十架武装飞行器的灯光照得如同白昼。
```

### 3. 意外转折
```
门打开了。
站在门外的，是林深以为已经死了十五年的父亲。
"好久不见，儿子。"
```

### 4. 悬念问题
```
照片上，年轻的赵医生和宋文博站在一起，
还有一个人——但那个人的脸，被人用刀划掉了。
是谁？
```

### 5. 倒计时
```
全息屏幕上，倒计时开始跳动：
23:59:58
二十四小时后，"创世纪"病毒将全球扩散。
```

### 6. 情感冲击
```
林雨闭上眼睛，呼吸越来越微弱。
心电监护仪的线条慢慢趋于平缓。
然后，一声长鸣。
```

## 📏 字数与结构

### 章节规格
- **总字数**：1800-2000字
- **场景数**：2-3个
- **开场**：50-100字（黄金三秒）
- **主体**：1600-1800字
- **结尾**：100-200字（钩子）

### 场景分配示例
```
场景1（600-700字）：开场冲突，抛出问题
场景2（600-700字）：冲突深化，推进剧情
场景3（600-700字）：转折或悬念，引向下章
```

## 🔍 常用描写模板

### 记忆当铺内部
```
霓虹招牌的冷光透过玻璃，照在[具体物品]上。
消毒水气味扑面而来，混合着[金属锈味/霉味]。
全息屏幕闪烁不定，在[破旧墙面]上投射[蓝色光斑]。
等候区[人数]个人，眼神空洞，像是[比喻]。
```

### 未来城市街景
```
霓虹灯海在雨夜[动作]，[颜色]的光晕[如何扭曲]。
悬浮广告播放着[内容]，标价[金额]。
高楼大厦上[企业logo]，楼下贫民窟[对比场景]。
街上人群[状态]，没人[动作]。
```

### 紧张场景
```
[声音1]。
[声音2]越来越近。
[人物]屏住呼吸。
[具体动作/物品]。
```

### 情绪低落
```
[人物]靠在[地点]，[外貌特征]，但[对比]依然[特征]。
"[台词]"
[人物]没说话。[TA]只是[动作]，
心里某个地方在[抽象动作]。
```

## 💡 实用技巧

### Show, Don't Tell
- ❌ Tell：他很穷
- ✅ Show：午餐唯一的蛋白质是合成淀粉块里的一只苍蝇

- ❌ Tell：她很强
- ✅ Show：她侧身避开激光刀，反手用探针刺入杀手麻筋

### 节奏控制
- **加快**：短句 + 动作 + 感官冲击
- **减慢**：长短句结合 + 环境描写 + 内心独白

### 伏笔技巧
- **违和感**：描写林深父亲回忆时暗示记忆被修改
- **应激反应**：苏晚对雨天和特定声音的头痛（伏笔车祸）
- **反常细节**：一尘不染的衣服上有几乎看不见的血迹
