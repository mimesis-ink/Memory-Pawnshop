## ✨ 核心特性

### 1. 完整的风格体系
- **赛博朋克**：高科技低生活、视觉对比、霓虹雨夜
- **黑色电影**：压抑冷峻、明暗对比、道德灰色
- **环境隐喻**：以景写情，细节暗示
- **语言克制**：简练有力，少即是多

### 2. 实用的写作工具
- **五种开场方式**：冲突、悬念、动作、对话、反常
- **六种结尾钩子**：真相、危机、转折、问题、倒计时、情感
- **环境描写模板**：记忆当铺、城市街景、紧张场景等
- **质量检查清单**：结构、风格、内容、技巧全覆盖

### 3. 分层的文档架构
- **深度学习**：writing_style_guide.md（512行详解）
- **流程指导**：chapter_writing_workflow.md（552行工作流）
- **快速查阅**：quick_reference.md（345行速查）
- **文档导航**：README.md（143行索引）

### 4. 一致的标准规范
- **字数**：1800-2000字/章
- **场景**：2-3个场景/章
- **开场**：50-100字黄金三秒
- **结尾**：100-200字悬念钩子
- **色调**：蓝、灰、黑冷色调
- **感官**：至少2种感官细节
