## 实施细节

### "黄金三秒"设计标准

**定义**：章节开头的前50-100字，必须在3秒内抓住读者注意力

**五种开场方式**：
1. ✅ 冲突开场：直接展示矛盾或危机
2. ✅ 悬念开场：抛出引人好奇的问题
3. ✅ 动作开场：从激烈的动作场景切入
4. ✅ 对话开场：用震撼性对话引入
5. ✅ 反常开场：展示违背常理的场景

**示例**（第1章）：
```
✅ 正确："支票上的血迹还是湿的。林深盯着那个500万的数字，
完全不记得自己为什么会有这张支票。"
```

### "结尾钩子"设计标准

**定义**：章节结尾的最后50-100字，必须制造强烈的期待感

**五种结尾方式**：
1. ✅ 悬念钩子：抛出新的疑问
2. ✅ 危机钩子：主角陷入危险
3. ✅ 反转钩子：意外的真相揭露
4. ✅ 情感钩子：情绪达到高潮
5. ✅ 预告钩子：暗示即将发生的大事

**示例**（第1章）：
```
✅ 正确："就在这时，林深的手机震动了一下。屏幕上是一条
匿名短信：'林深，你以为卖掉记忆就安全了吗？48小时内，
他们会来灭口。'"
```
