# 章节创作指南

## 项目体量规范

### 总体规格
- **总字数**：20-25万字
- **章节数量**：100-120章
- **单章字数**：1800-2000字
- **创作节奏**：短剧节奏，快速推进

### 核心创作原则

本项目采用**短剧节奏**创作方式，每章必须严格遵循以下两大核心要素：

#### 1. "黄金三秒"（Golden Three Seconds）
**定义**：章节开头的前50-100字，必须立即抓住读者注意力

**目的**：
- 在3秒内吸引读者继续阅读
- 制造悬念或视觉冲击
- 建立情绪基调

**实现方法**：
- ✅ **冲突开场**：直接展示矛盾或危机
- ✅ **悬念开场**：抛出引人好奇的问题
- ✅ **动作开场**：从激烈的动作场景切入
- ✅ **对话开场**：用震撼性对话引入
- ✅ **反常开场**：展示违背常理的场景

**示例**：
```
❌ 错误示例：
"林深这天早上像往常一样起床，洗漱完毕后吃了早餐..."

✅ 正确示例：
"支票上的血迹还是湿的。林深盯着那个500万的数字，完全不记得自己为什么会有这张支票。"
```

#### 2. "结尾钩子"（Ending Hook）
**定义**：章节结尾的最后50-100字，必须制造强烈的期待感

**目的**：
- 让读者迫不及待想看下一章
- 留下悬念或未解之谜
- 强化情绪或冲突

**实现方法**：
- ✅ **悬念钩子**：抛出新的疑问
- ✅ **危机钩子**：主角陷入危险
- ✅ **反转钩子**：意外的真相揭露
- ✅ **情感钩子**：情绪达到高潮
- ✅ **预告钩子**：暗示即将发生的大事

**示例**：
```
❌ 错误示例：
"林深回到家，洗了个澡就睡觉了。"

✅ 正确示例：
"就在林深准备离开时，手机震动了一下。屏幕上是一条匿名短信：'你以为卖掉记忆就安全了吗？他们已经盯上你妹妹了。'"
```

## 章节创作清单

每完成一章，请确认以下要点：

### 内容要求
- [ ] 字数在1800-2000字之间
- [ ] 包含"黄金三秒"开场（50-100字）
- [ ] 包含"结尾钩子"结尾（50-100字）
- [ ] 至少包含2-3个场景转换
- [ ] 推进主线剧情或揭示关键信息
- [ ] 符合短剧节奏，避免拖沓

### 节奏要求
- [ ] 开场立即抓住注意力
- [ ] 中间保持紧张感和推进感
- [ ] 结尾制造悬念或期待
- [ ] 没有冗长的描写或解释
- [ ] 对话精炼有力

### 世界观一致性
- [ ] 符合2087年未来世界设定
- [ ] 记忆技术描述准确
- [ ] 角色行为符合性格设定
- [ ] 情节逻辑自洽

## 章节分类指南

### 按功能分类

#### 1. 情节推进章（60-70%）
**作用**：推动主线剧情发展
**特点**：
- 揭示新的线索或信息
- 推进主角目标
- 制造新的冲突

#### 2. 情感铺垫章（15-20%）
**作用**：深化角色关系和情感
**特点**：
- 角色互动和对话
- 情感积累
- 为后续爆发做准备

#### 3. 悬念揭晓章（10-15%）
**作用**：解开悬念，制造反转
**特点**：
- 真相揭露
- 意外反转
- 强烈冲击感

#### 4. 高潮章节（5-10%）
**作用**：重大事件和转折点
**特点**：
- 激烈冲突
- 重大决定
- 情绪爆发

## 短剧节奏设计原则

### 1. 快节奏推进
- **每章必须有新进展**：不能原地踏步
- **避免冗长铺垫**：信息密度要高
- **场景切换迅速**：2-3个场景为宜

### 2. 密集悬念设计
- **层层设问**：每解开一个谜题，引出新的谜题
- **多线并进**：主线+副线+伏笔同时推进
- **节奏变化**：紧张-缓和-紧张的波浪式节奏

### 3. 强烈视觉感
- **画面感强**：便于影视化改编
- **动作描写**：具体、有冲击力
- **场景鲜明**：突出赛博朋克美学

## 100-120章结构规划

### 第一幕：开局与危机（第1-30章）
**核心任务**：建立世界观，引入主要矛盾

- **第1-5章**：林深卖掉记忆，发现异常
- **第6-10章**：调查开始，遇见苏晚
- **第11-15章**：进入记忆黑市，线索初现
- **第16-20章**：威胁升级，妹妹陷入危险
- **第21-25章**：苏晚身份揭晓
- **第26-30章**：第一次记忆碎片恢复

### 第二幕：对抗与追寻（第31-80章）
**核心任务**：深入调查，揭露真相，情感升华

- **第31-40章**：陈煜势力追杀，开始逃亡
- **第41-50章**：记忆拼图，真相逐渐浮现
- **第51-60章**：林深与苏晚过往揭露
- **第61-70章**：赵医生的阴谋曝光
- **第71-80章**：深入敌后，寻找关键证据

### 第三幕：高潮与结局（第81-120章）
**核心任务**：最终对决，真相大白，情感升华

- **第81-90章**：记忆完全恢复，真相震撼
- **第91-100章**：最终对决准备
- **第101-110章**：正面交锋，揭露陈煜
- **第111-115章**：危机解除，代价显现
- **第116-120章**：尾声，开放式结局

## 创作技巧

### 1. 开场技巧
- **从冲突开始**：不要从平静开始
- **制造反差**：用反常引发好奇
- **视觉冲击**：用强烈画面吸引注意

### 2. 中段技巧
- **信息递进**：每个场景都有新信息
- **情绪起伏**：制造情绪的波动
- **伏笔埋设**：为后续章节做铺垫

### 3. 结尾技巧
- **未解之谜**：留下疑问
- **危机升级**：让处境更危险
- **反转预告**：暗示意外即将发生

## 常见问题

### Q1: 如果一个场景超过2000字怎么办？
A: 将场景拆分成多章，每章聚焦一个关键节点，确保每章都有独立的"黄金三秒"和"结尾钩子"。

### Q2: 如何平衡信息量和节奏？
A: 采用"展示而非讲述"的原则，通过对话和行动传递信息，避免大段说明性文字。

### Q3: 每章都要有反转吗？
A: 不需要。但每章必须有"新进展"——新信息、新冲突、新情感，让读者感觉到故事在前进。

### Q4: 如何控制字数精确度？
A: 
- 主体内容控制在1600-1800字
- 黄金三秒50-100字
- 结尾钩子50-100字
- 总计约1800-2000字

### Q5: 情感章节也需要快节奏吗？
A: 是的。即使是情感章节，也要避免冗长对白，通过简洁有力的互动展现情感，保持短剧节奏。

## 时间线一致性

### 参考时间线文档
创作新章节时，**必须参考**[故事时间线](timeline.md)确保时间连贯性。

### 时间标注规范
每章开头应明确标注：
- **具体日期**：例如"12月18日（周一）"
- **具体时间**：例如"上午9:00"或"深夜23:30"
- **时间跨度**：如果章节跨越多个时段，需明确标注

### 时间线更新流程
1. **创作前**：查阅timeline.md，确定章节时间点
2. **创作中**：遵循已定时间顺序，避免矛盾
3. **创作后**：将新增关键事件更新到timeline.md

### 示例
```markdown
# 第25章：威胁降临

**时间**：12月20日（周三）09:00  
**地点**：林深家

## 【黄金三秒】
门铃响了三次，林深透过猫眼看到两个穿黑西装的男人...
```
