# 项目规格说明书

## 项目概述

**项目名称**：《记忆当铺：我卖掉了杀你的记忆》  
**英文名称**：Memory Pawnshop: I Sold the Memory of Killing You  
**类型**：科幻悬疑短剧/网络小说  
**状态**：开发中

## 体量规格

### 版本一：短剧版
- **集数**：12集
- **单集时长**：约15分钟
- **总时长**：约180分钟（3小时）
- **格式**：影视剧本（场景、对话、镜头描述）
- **适用场景**：影视制作、视频平台播出

### 版本二：章节版（主要版本）
- **章节数量**：100-120章
- **单章字数**：1800-2000字（严格）
- **总字数**：200,000-250,000字（20-25万字）
- **格式**：短剧节奏网文
- **适用场景**：网络连载、电子书出版、短视频改编

## 核心创作规范

### "黄金三秒"（Golden Three Seconds）

**定义**：每章开头的前50-100字，必须在3秒内抓住读者注意力。

**实现标准**：
- ✅ **字数**：50-100字
- ✅ **位置**：章节开头，独立段落，用`【黄金三秒】`标记
- ✅ **效果**：立即制造冲突、悬念、好奇或震撼
- ✅ **禁止**：平淡叙事、背景描写、流水账

**五种开场方式**：
1. **冲突开场**：直接展示矛盾或危机
2. **悬念开场**：抛出引人好奇的问题
3. **动作开场**：从激烈的动作场景切入
4. **对话开场**：用震撼性对话引入
5. **反常开场**：展示违背常理的场景

**质量检测**：
- [ ] 能否在3秒内让读者产生"想知道接下来发生什么"的冲动？
- [ ] 是否避免了"早上起床""天气很好"等平淡开场？
- [ ] 是否立即切入核心冲突或悬念？

### "结尾钩子"（Ending Hook）

**定义**：每章结尾的最后50-100字，必须制造强烈的期待感，让读者迫不及待看下一章。

**实现标准**：
- ✅ **字数**：50-100字
- ✅ **位置**：章节结尾，独立段落，用`【结尾钩子】`标记
- ✅ **效果**：制造悬念、危机、反转或强烈情绪
- ✅ **禁止**：完美收尾、情节完全解决、平淡结束

**五种结尾方式**：
1. **悬念钩子**：抛出新的疑问，未解之谜
2. **危机钩子**：主角陷入危险，处境恶化
3. **反转钩子**：意外的真相揭露，出乎意料
4. **情感钩子**：情绪达到高潮，戛然而止
5. **预告钩子**：暗示即将发生的大事

**质量检测**：
- [ ] 读者是否会立即想要点开下一章？
- [ ] 是否避免了"回家睡觉""问题解决"等完结式结尾？
- [ ] 是否留下足够的悬念或期待？

## 创作节奏要求

### 短剧节奏三原则

#### 1. 快速推进
- **每章必须有新进展**：情节推进、线索揭露、冲突升级
- **避免冗长铺垫**：信息密度要高，无用对话要删
- **场景切换迅速**：每章2-3个场景为宜，不超过4个

#### 2. 密集悬念
- **层层设问**：每解开一个谜题，引出新的谜题
- **多线并进**：主线+副线+伏笔同时推进
- **节奏变化**：紧张-缓和-紧张的波浪式节奏

#### 3. 强烈视觉感
- **画面感强**：便于影视化改编
- **动作描写**：具体、有冲击力、避免抽象
- **场景鲜明**：突出赛博朋克美学和记忆可视化

## 100-120章整体规划

### 第一幕：开局与危机（第1-30章）
**字数**：约54,000-60,000字  
**核心任务**：建立世界观，引入主要矛盾

**章节分配**：
- 第1-5章：林深卖掉记忆，发现血色支票
- 第6-10章：调查开始，遇见苏晚
- 第11-15章：进入记忆黑市，线索初现
- 第16-20章：威胁升级，妹妹陷入危险
- 第21-25章：苏晚身份揭晓，联手调查
- 第26-30章：第一次记忆碎片恢复

### 第二幕：对抗与追寻（第31-80章）
**字数**：约90,000-100,000字  
**核心任务**：深入调查，揭露真相，情感升华

**章节分配**：
- 第31-40章：陈煜势力追杀，开始逃亡
- 第41-50章：记忆拼图，真相逐渐浮现
- 第51-60章：林深与苏晚过往揭露
- 第61-70章：赵医生的阴谋曝光
- 第71-80章：深入敌后，寻找关键证据

### 第三幕：高潮与结局（第81-120章）
**字数**：约72,000-80,000字  
**核心任务**：最终对决，真相大白，情感升华

**章节分配**：
- 第81-90章：记忆完全恢复，真相震撼
- 第91-100章：最终对决准备，集结盟友
- 第101-110章：正面交锋，揭露陈煜罪行
- 第111-115章：危机解除，代价显现
- 第116-120章：尾声，开放式结局，情感升华

## 章节分类与占比

### 按功能分类

1. **情节推进章（60-70%，约60-84章）**
   - 作用：推动主线剧情发展
   - 特点：揭示新线索、推进目标、制造新冲突

2. **情感铺垫章（15-20%，约15-24章）**
   - 作用：深化角色关系和情感
   - 特点：角色互动、情感积累、为爆发做准备

3. **悬念揭晓章（10-15%，约10-18章）**
   - 作用：解开悬念，制造反转
   - 特点：真相揭露、意外反转、强烈冲击

4. **高潮章节（5-10%，约5-12章）**
   - 作用：重大事件和转折点
   - 特点：激烈冲突、重大决定、情绪爆发

## 创作流程

### 1. 章节规划阶段
- [ ] 确定本章在整体结构中的位置（第几章，第几幕）
- [ ] 明确本章的核心任务（推进情节/深化情感/揭露真相/制造高潮）
- [ ] 列出本章需要完成的3-5个关键要素
- [ ] 设计"黄金三秒"和"结尾钩子"

### 2. 初稿创作阶段
- [ ] 先写"黄金三秒"，确保立即抓人
- [ ] 按场景展开主体内容（2-3个场景）
- [ ] 最后写"结尾钩子"，制造悬念
- [ ] 检查字数（1800-2000字）

### 3. 修改优化阶段
- [ ] 删除冗余对话和描写
- [ ] 强化视觉感和画面感
- [ ] 检查逻辑自洽性
- [ ] 优化"黄金三秒"和"结尾钩子"
- [ ] 最终字数控制在1800-2000字

### 4. 质量检查阶段
- [ ] 对照"优秀章节标准"逐项检查
- [ ] 确认没有"需改进的信号"
- [ ] 请他人试读，收集反馈
- [ ] 根据反馈进行最后调整

## 技术规范

### 文件格式
- **文件类型**：Markdown (.md)
- **编码**：UTF-8
- **换行**：LF (Unix)

### 文件命名
- **格式**：`chapters_XX_YY.md`
- **示例**：`chapters_01_05.md`（第1-5章）
- **规则**：小写字母，下划线分隔，两位数字

### 章节标题
- **格式**：`# 第X章：章节标题`
- **示例**：`# 第1章：血色支票`
- **规则**：使用中文数字或阿拉伯数字，标题简洁有力

### 特殊标记
- **黄金三秒**：`## 【黄金三秒】`
- **结尾钩子**：`## 【结尾钩子】`
- **场景分隔**：`### 场景X：场景名称`
- **章节分隔**：`---`（三个短横线）
