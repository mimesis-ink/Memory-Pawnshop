# 记忆当铺：我卖掉了杀你的记忆

## 项目简介

这是一部15集悬疑科幻短剧剧本，讲述了一个关于记忆买卖、贫富差距和自我救赎的故事。

**核心概念**："贫穷限制了想象力，也限制了你拥有美好记忆的权利。"

**流量密码**：贫富差距 + 悬疑探案 + 爽文反转

---

## 故事梗概

未来世界，记忆可以像器官一样买卖。穷人卖掉快乐换钱，富人购买快乐填补空虚。

男主林墨是一名"记忆提取师"，为了救重病的妹妹，他不得不卖掉自己最珍贵的一段初恋记忆。然而手术后他发现，自己口袋里有一张带血的巨额支票，而所有线索都指向——他刚刚卖掉的记忆，并不是"初恋"，而是他"目击了一场针对顶级富豪的谋杀"。

现在，富豪要来灭口，而他必须找回那段被删除的记忆才能活命。

## 核心特色

### ✅ 开篇即高能（黄金3秒）
第一集前3秒抛出核心悬念：醒来发现口袋里有血和500万支票

### ✅ 视觉奇观落地
科幻元素道具化：
- 后脑记忆接口（LED发光）
- 发光的记忆存储管
- 记忆拍卖会场景

### ✅ 情绪价值优先
- **亲情线**：林墨为救妹妹不惜牺牲一切
- **正义线**：揭露记忆黑市，伸张正义
- **成长线**：从被动受害者到主动行动者

### ✅ 爽文反转
- 第6集：买家竟是死者本人？
- 第15集：双胞胎其实是人格分裂，自我谋杀的终极反转
