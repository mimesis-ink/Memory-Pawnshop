## 项目简介

**书名**：《记忆当铺：我卖掉了杀你的记忆》（Memory Pawnshop: I Sold the Memory of Killing You）

**类型**：科幻悬疑小说  
**章节数**：90章（每章1800-2000字）  
**总字数**：约18万字  
**核心概念**："贫穷限制了想象力，也限制了你拥有美好记忆的权利。"

---

## 故事梗概

未来世界，记忆可以像器官一样买卖。穷人卖掉快乐换钱，富人购买快乐填补空虚。

男主林深是一名"记忆提取师"，为了救重病的妹妹，他不得不卖掉自己最珍贵的一段初恋记忆。然而手术后他发现，自己口袋里有一张带血的巨额支票，而所有线索都指向——他刚刚卖掉的记忆，并不是"初恋"，而是他"目击了一场针对顶级富豪的谋杀"。

现在，富豪要来灭口，而他必须找回那段被删除的记忆才能活命。

## 主要人物

### 主角阵营

- **林深**（男主，28岁）：记忆提取师（5年从业经验），为救绝症妹妹卖掉记忆，后卷入庞大阴谋。他不知道自己是记忆技术创始人周明的亲生儿子，卖掉的"初恋记忆"实为目击谋杀，口袋里有带血的500万支票。苏晚的男朋友（5年前相识相爱，因记忆交易失去彼此记忆，现在逐渐重建感情）
- **苏晚**（女主，26岁）：记忆当铺调查员，实为公安局卧底调查记忆黑市。5年前车祸后大脑受损，购买过记忆碎片修复神经（实际上是林深救她的记忆）。林深的女朋友，与林深共同战斗，干练能打，但因记忆拼凑而常自我怀疑
- **林雨**（妹妹，18岁）：患神经退行性疾病的大学生（艺术系），需要500万手术费，林深的道德锚点和动力源泉。善良坚强，代表纯真和希望

### 反派阵营（层级森严）

- **周明**（终极BOSS，60岁）：林深的生父（化名林建国），记忆技术真正的发明者，宋文博和李教授的导师。2072年假死，幕后操控一切。创立"深渊计划"和记忆当铺，是整个阴谋的源头，旨在实现意识永生。第85-90章揭露真相
- **韩世杰**（幕后黑手，70岁）：记忆产业联盟创始人，真正的大反派。控制合法市场30%GDP，黑市60-70%，核心技术接近100%垄断。绝对冷酷，对亲生儿子韩江开枪。掌握政界高层、财阀巨富、军方将领的记忆秘密
- **陈煜**（初期反派，45岁）：记忆产业集团董事长，个人资产超1000亿，实际受韩世杰控制。冷酷狡猾，拥有多个克隆体。第10章自杀，但克隆体继续出现
- **李明**（中期反派，30岁）：永生会表面创始人（实际被周明利用的棋子），记忆技术专家。因妹妹李婉2077年死于技术不成熟而扭曲，追求"全员永生/意识上传"，推行"创世纪"AI计划。拥有上百个记忆克隆备份。五年前被韩世杰从监狱保释招募
- **张立军**（悲剧傀儡，52岁）：市公安局局长，从警28年。五年前被植入虚假记忆（以为自己是永生会创始人"老师"），实为替罪羊。第50章自杀，临死前试图留下关于"第八号岛"的线索

### 关键摇摆人物与盟友

- **赵医生/赵文渊/赵启明**（复杂角色，50岁）：记忆当铺主任医师，从业20年。与陈煜是联合创始人，知晓最多技术内幕。后期分裂为三个版本（原版、被控制版、数字意识版），第60章忏悔，第69章做出牺牲性选择
- **韩江**（重要盟友，42岁）：韩世杰的儿子，地下黑市记忆医生。发现父亲罪行后背叛，帮助林深等人。第24章被父亲开枪射伤。第9-85章持续出现，是团队的技术支持核心
- **沈微**（技术专家，28岁）：宋文博的学生，林深的研究伙伴。出身军人家庭，熟练枪械格斗。在宋文博被杀后保护技术资料和林深
- **宋文博**（已故，45岁时去世）：记忆溯源技术发明者，2087年12月17日被韩世杰枪杀。身体死亡但意识/记忆转移保存，后期以数字形式存在
- **李婉**（已故，享年12岁）：李明的妹妹，2077年死于神经退行性脑部疾病。临终要求李明"不要为救她而伤害他人"，但李明最终违背。她的死是李明走向疯狂的起点，与林雨形成悲剧对比

## 核心主题

1. **贫富差距**：记忆买卖加剧社会不平等
2. **身份认同**：失去记忆后，我们还是自己吗？
3. **科技伦理**：技术进步带来的道德挑战
4. **真情可贵**：可以买卖一切的世界，真情才最珍贵
