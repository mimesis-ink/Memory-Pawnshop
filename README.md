## 项目简介

**剧名**：《记忆当铺：我卖掉了杀你的记忆》（Memory Pawnshop: I Sold the Memory of Killing You）

**类型**：科幻悬疑小说  
**章节版**：90章（每章1800-2000字）  
**总字数**：约18万字  
**核心概念**："贫穷限制了想象力，也限制了你拥有美好记忆的权利。"

---

## 故事梗概

未来世界，记忆可以像器官一样买卖。穷人卖掉快乐换钱，富人购买快乐填补空虚。

男主林深是一名"记忆提取师"，为了救重病的妹妹，他不得不卖掉自己最珍贵的一段初恋记忆。然而手术后他发现，自己口袋里有一张带血的巨额支票，而所有线索都指向——他刚刚卖掉的记忆，并不是"初恋"，而是他"目击了一场针对顶级富豪的谋杀"。

现在，富豪要来灭口，而他必须找回那段被删除的记忆才能活命。

---

## 主要人物

- **林深**（男主，28岁）：记忆提取师，为救妹妹卖掉记忆，后卷入庞大阴谋
- **苏晚**（女主，26岁）：记忆当铺调查员，实为卧底警察，与林深共同战斗
- **林雨**（妹妹，18岁）：重病学生，林深的动力源泉
- **周明**（最终反派，55岁）：林深的父亲，记忆技术创始人，整个阴谋的源头
- **陈煜**（主要反派，45岁）：记忆产业巨头，拥有多个克隆体
- **赵医生/赵启明**（复杂角色，50岁）：记忆当铺医师，存在三个版本，经历觉醒与救赎
- **李明**（反派，30岁）：永生会创始人，推行"创世纪"AI计划
- **韩世杰**（警局长，50岁）：双重身份，与永生会有复杂关联

---

## 核心主题

1. **贫富差距**：记忆买卖加剧社会不平等
2. **身份认同**：失去记忆后，我们还是自己吗？
3. **科技伦理**：技术进步带来的道德挑战
4. **真情可贵**：可以买卖一切的世界，真情才最珍贵
